# Rotation Web

React 版のローテーション入力サイトです。

## 起動

```bash
npm install
npm run dev
```

ブラウザで表示された URL を開くと、名前ごとに `○ / △ / × / 総会のみ` を入れられます。

## できること

- 名前を選んで自分の出席を入力
- 入力内容をブラウザの `localStorage` に自動保存
- 5月の各回について自動で担当を決定
- `△` は人数不足のときだけ採用
- 王さん参加週は人数が足りれば入門を2クラスに分割

## 注意

- 今の実装は `localStorage` 保存なので、同じブラウザ・同じ端末の中で使う前提です。
- 複数人が別々の端末から同じデータを共有するには、あとでバックエンドが必要です。
